# ⏳ Time Capsule System

Ek modern web application jo users ko mauka deta hai apne future ke liye messages aur yaadein schedule karne ka. Yeh project Node.js aur Express par banaya gaya hai.

## 🌟 Key Features
* **Message Scheduling:** Apni pasand ki date aur time par messages schedule karein.
* **Auto-Unlock Logic:** Messages tab tak "Locked" rahenge jab tak unka sahi waqt nahi aa jata.
* **Professional Dashboard:** Ek clean aur responsive UI jahan aap apne saare capsules dekh sakte hain.
* **Notifications:** Naye messages ke liye real-time alerts aur badges.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Design), JavaScript (ES6+)
* **Backend:** Node.js, Express.js
* **API Routing:** Express Router for clean code structure.
* **Environment:** `.env` for secure configuration.

## 📂 Folder Structure
* `public/` - Saari frontend files (HTML, CSS, JS).
* `routes/` - Backend endpoints aur logic.
* `config/` - Database aur app settings.
* `server.js` - Main entry point of the application.

## 🚀 How to Run
1. Project ko clone karein.
2. `npm install` chala kar dependencies install karein.
3. `.env` file set karein.
4. `npm start` se server shuru karein.
